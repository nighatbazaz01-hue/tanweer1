## 4. Key Design Principles
*   **Clarity over Decoration:** High data density required for ERP, but must remain clean and readable.
*   **Accessibility First:** Ensure WCAG compliance to support all user types (Staff, Students, Parents).
*   **Consistency:** Use the established spacing and color tokens to maintain a unified platform feel.
*   **AI-Integrated:** The AI Assistant should feel like a native, always-available extension of the UI.

## 5. Technical Stack Integration
- **Framework:** Next.js.
- **Styling:** Tailwind CSS (configured with design system tokens).
- **Primitives:** Shadcn UI + Radix UI.
- **Icons:** Lucide React.
