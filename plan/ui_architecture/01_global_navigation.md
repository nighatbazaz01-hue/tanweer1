## 2. Navigation Hierarchy

### Top Navigation (Global Actions)
*   School Selector (for Super Admin)
*   Academic Year Switcher
*   Global Search (Students, Staff, Books)
*   Notifications Bell
*   AI Assistant Quick-launch
*   User Profile (Settings, Logout)

### Sidebar Navigation (Modular)
The sidebar is dynamic based on user roles and permissions.

*   **Admin/Principal Sidebar:**
    *   Dashboard
    *   Admissions (Leads, Applications)
    *   Students (Profiles, Attendance)
    *   Academic (Classes, Timetable)
    *   Finance (Fees, Expenses)
    *   HR (Staff, Payroll)
    *   AI Insights
    *   Settings

*   **Teacher Sidebar:**
    *   My Dashboard
    *   My Classes
    *   Attendance Marking
    *   Assignments
    *   Exams & Grading
    *   Timetable
    *   AI Teacher Assistant

*   **Parent/Student Sidebar:**
    *   Dashboard
    *   My Profile / My Children
    *   Daily Attendance
    *   Homework & Tasks
    *   Exam Schedule & Results
    *   Fee Payments
    *   Transport Tracking
    *   AI Assistant

## 3. Global Layout Design
*   **Sidebar:** Collapsible, dark theme for professional focus.
*   **Topbar:** Sticky, light theme, contains breadcrumbs and global actions.
*   **Main Content:** Card-based layout, white background, ample whitespace.
*   **AI Assistant:** Floats in bottom right, expands to right-side drawer.
