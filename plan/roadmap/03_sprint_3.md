# Tanweer Development Roadmap: Sprint 3 (Weeks 5-6)

## Goal
Establish the academic framework including classes, staff, timetable, and attendance.

## Features
*   Staff/HR records and Subject assignments.
*   Class and Section management.
*   Timetable builder.
*   Daily Student and Staff attendance.

## Database Work
*   Implement `staff`, `classes`, `sections`, `subjects`.
*   Implement `timetable_entries`, `attendance_records`.

## Backend Work
*   Implement Academic module (Classes/Subjects).
*   Implement Timetable engine with conflict detection.
*   Implement Attendance module.

## Frontend Work
*   Build Timetable grid and builder interface.
*   Create Attendance marking sheets for Teachers.
*   Implement Staff/HR management directory.

## Testing
*   Validation tests for timetable conflict detection.
*   Load testing for bulk attendance submission.
*   UAT with "Teacher" role for attendance and timetable viewing.

## Deliverables
*   Functional weekly timetable for classes.
*   Digital attendance marking system.
*   Complete staff/teacher directory.
