## PIN Accounts

Session-level PIN gate protecting sensitive student fields (financials, disciplinary records, contact details).
- 3 failed attempts → account locked, admin notified via `useDataStore.addNotification`
- 15-minute lockout enforced in `src/store/usePinStore.ts`
- `getPinForRole(appRole)` in `src/lib/mockData/credentials.ts` is the lookup function

| Role | User | PIN |
|------|------|-----|

| vp2 | Ms. Nora Al-Zahrani | `9012` |
| vp3 | Mr. Faris Al-Mutairi | `3456` |

> Teacher, Parent, and Student roles do not have PINs — they cannot access PIN-gated views.
